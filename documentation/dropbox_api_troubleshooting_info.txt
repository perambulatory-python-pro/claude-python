files_list_folder return type:
In the Dropbox API Python SDK, the files_list_folder method returns a ListFolderResult object. 
This ListFolderResult object contains a list of entries, including file and folder metadata, in its entries attribute. Each entry within entries is of type Metadata, and for files specifically, they are of type FileMetadata. 
Important Notes:
The files_list_folder method is paginated, meaning it may not return all entries in a single call.
You must check the has_more attribute of the returned ListFolderResult. If has_more is True, you need to call files_list_folder_continue with the cursor from the previous result to retrieve the remaining entries.
